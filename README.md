## About BlogHub

**BlogHub** is a modern web application built using the Laravel framework. It provides a platform for users to share, comment, and engage with blog posts. **BlogHub** emphasizes ease of use, user-friendly navigation, and seamless performance, making it an ideal solution for content creators and readers.

### Key Features of BlogHub:
- User-friendly blog post creation and management.
- Secure authentication using Laravel Breeze.
- Full support for comments, likes, and user interaction.
- Admin panel for managing posts and categories.
- Real-time notifications using Laravel Broadcasting.
- Responsive design with Tailwind CSS for mobile-friendly use.

## Tech Stack

- **Backend**: Laravel 11 (PHP) with Laravel Breeze for authentication.
- **Frontend**: Tailwind CSS and Trix Editor for content creation.
- **Database**: MySQL, integrated with Laravel's Eloquent ORM for managing blog content and user data.
- **Storage**: Images and media files are stored in the `storage/image` directory.

## BlogHub Features

### For Users:
- View blog posts.
- Comment on posts and interact with likes.
- User profile management (edit bio, update profile picture, etc.).
- Role-based system: Regular users and Admin.
- Can See select post by category

### For Admins:
- Create, edit, and delete blog posts.
- Manage blog categories.
- View user activity and moderate comments.
- CRUD operations on user roles, posts, and categories.

## Laravel Features Used in BlogHub:
- [Simple, fast routing engine](https://laravel.com/docs/routing) to handle user requests efficiently.
- [Dependency injection container](https://laravel.com/docs/container) for managing application services.
- [Database migrations](https://laravel.com/docs/migrations) for version-controlled schema updates.
- [Real-time broadcasting](https://laravel.com/docs/broadcasting) for notifications and interactions.
